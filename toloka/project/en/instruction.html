Annotation guideline

The quality of the annotation is strictly controlled. New assignments will not be issued to dishonest or inaccurate users.

The task is to determine the relationship between two news titles.

Two titles are called the same if they are about the same things or differ from each other in minor details.

The first title causes the second title if the second title is impossible without the first one. That is, if the first event didn't happen, then the second event should not be happining too. Examples: opinions, details, comments, true causes. This relationship should be ovbious from the titles only.

The second title refutes the first one if the second title makes the first one irrelevant. Examples: refutations, updates in death tool, updates in match score. This relationship should be ovbious from the titles only.

Other possible relationships include the same characters, titles with the same cause, titles with unknown order. This relationship should be chosen in case if you are in doubt.

Overall, there are 7 possible answers:
  1. Same titles
  2. B refutes A
  3. A refutes B
  4. A causes B
  5. B causes A
  6. Other relationship
  7. No relationship

Examples

#1
A: 11
B: 11
Result: A causes B
